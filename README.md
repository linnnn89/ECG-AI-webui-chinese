# ECG-AI WebUI Chinese

ECG-AI WebUI Chinese is a local research-oriented web application for ECG software workflow exploration.

This top-level README is written in simple English for journal reviewers, international readers, and non-Chinese developers who need a quick overview of the repository.

The detailed documentation is currently written in Chinese and is located in the secondary project directory:

[ECG AI project/README.md](ECG%20AI%20project/README.md)

For non-Chinese readers, we recommend using a large language model (LLM) to translate and read the Chinese README. The Chinese README contains more complete information about installation, directory structure, data preparation, inference workflow, image processing, safety behavior, troubleshooting, and future development plans.

## Main features

- Local WebUI for ECG AI workflow exploration.
- WFDB-based ECG inference.
- ECG XML conversion and inference.
- Experimental ECG image workflow.
- ONNX-based inference backend.
- Multi-label output.
- Chinese report generation workflow.
- Similar-case retrieval design for local use.
- Uncertainty handling for low-confidence results.
